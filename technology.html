Technical Stack

We use blockchain

  • The system is highly using MIOTA and saving data in blockchain to make sure sensitive data is highly private and secured.

    • Private Blockchain server (Collection of private node, spread around the world between many cloud providers)
    • Blockchain network get updated versions of the ledger as new transactions are made
    • The multiple copies of the ledger are the “truth” about every transaction made so far in the blockchain.
    • Consensus - All the connected node in the chain agree that each transaction is valid.
    • Immutability - No node can tamper with an entry in the distributed ledger. (Like Bitcoin)

Its on most secure cloud

  • Google Cloud

    The system is based on Google cloud. We have used several components of Google cloud to increase efficiency of the application. We use firebase and GC SQL primarily.

We are using most efficient languages

  • We highly rely on Node, Python and Java for server and client applications. Same time we could not avoid few services like Google pub sub etc for queuing and other services.

Messaging and queing

  • The application uses Google Pub Sub for highly reliable messaging and queuing service.

This is Smart

  • The application uses Google machine learning. This is to protect theft or any misbehavior by training the machine(regression). We also track personalized kids behavior and generate alert on any irregular behavior.
